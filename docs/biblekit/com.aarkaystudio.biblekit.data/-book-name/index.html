BookName

enum BookName : Enum<BookName>

Represents the canonical books of the Bible in their traditional order. This enum provides both full names and standardized abbreviations for all 66 books, divided into the Old Testament (Genesis through Malachi) and New Testament (Matthew through Revelation).

Each book has:

  • A full name (e.g., "Genesis", "1 Corinthians")

  • A standardized abbreviation (e.g., "Gen", "1Co")

  • An implicit ordinal position (1-66) based on its declaration order
